阶段一 · 网页问答 把 AI 当高级搜索引擎用
2021 年 6 月
GitHub Copilot 技术预览上线 首发
OpenAI Codex 驱动，仅限 VS Code，代码补全首次进入大众视野。开始有人把代码粘贴进对话框问 AI。
2022 年 6 月
GitHub Copilot 正式对外开放 GA
首月新增 40 万付费用户。标志着 AI 辅助编程开始规模化，但交互仍是 IDE 之外的"复制粘贴"方式。
2022 年 11 月
ChatGPT 上线（2022.11.30）里程碑
5 天 100 万用户，两个月 1 亿用户，史上最快增长的消费应用。开发者开始大规模用聊天界面写代码、问问题——"网页问答"模式爆发。
2023 年 3 月
GPT-4 发布 + ChatGPT API 开放 模型
能力跃升，开发者涌入。大量产品开始接入 API，但交互范式仍是"问答"——人问 AI 答，代码自己粘贴。
阶段二 · IDE 插件内联辅助 AI 进入编辑器，能"看见"你的代码
2023 年 3 月
Cursor 发布（基于 VS Code fork）首发
Anysphere 四位 MIT 毕业生创立。AI 原生 IDE，Chat 与代码并排，首个真正"嵌入项目"的 AI 编码体验。到当年 11 月已索引 15 万个代码库。
2023 年下半年
Continue、通义灵码等陆续涌现 工具潮
IDE 插件赛道爆发。GitHub Copilot Chat 上线，国内通义灵码、文心快码相继推出。行业共识：AI 要进编辑器，有代码上下文才有用。
2024 年 6 月
Cursor 完成 6000 万美元 A 轮 融资
a16z 领投，验证"AI 原生 IDE"方向。Cursor 引入 Composer 模式——首个支持多文件编辑的主流 IDE 插件，开始向 Agent 过渡。
阶段三 · 编程 Agent 自主执行 AI 从"建议者"变为"执行者"
2024 年 3 月 12 日
Devin 发布——首个"AI 软件工程师" 范式转移
Cognition Labs 发布，Demo 视频在 X 上获 3000 万次播放。首次让外界意识到 AI 可以自主完成完整工程任务：写代码 → 运行 → 报错 → 修复，全自动循环。
2025 年 2 月
Claude Code 研究预览上线 首发
Anthropic 发布，终端原生 Agent，可读写文件、执行命令、提交 Git。核心差异：不是 IDE 插件，而是直接驻留在终端里的 Agent。
2025 年 5 月
Claude Code GA + Claude 4 发布 GA模型
正式商用，支持 GitHub Actions 后台任务、VS Code / JetBrains 原生集成。Opus 4 在 SWE-bench 达到 72.5%。Agent 编程进入主流。
2025 年 11 月
Claude Code 年化营收突破 10 亿美元 数据
发布 6 个月内破 10 亿，2026 年初逼近 20 亿。Cursor 同月年化营收突破 10 亿，估值 293 亿美元。Agent 赛道全面商业化。
阶段四 · Harness 工程 驾驭 Agent，设计让 AI 可靠工作的系统
2025 年 8 月
AGENTS.md 规范发布 规范
OpenAI、Google、Cursor、Factory 等联合制定，Agent 跨工具行为约束文件的开放标准，Harness 工程的基础设施开始标准化。
2025 年 12 月
Karpathy 提出"上下文工程"概念 概念
Andrej Karpathy 在 X 发文，区分 Prompt Engineering 与 Context Engineering，引发工程师重新审视"如何给 Agent 提供信息"的问题。
2026 年 2 月 11 日
OpenAI 正式定义"Harness 工程" 里程碑
Ryan Lopopolo 发文，记录以零手写代码交付百万行生产应用的经验，提出 Agent = Model + Harness，Humans steer, Agents execute。Mitchell Hashimoto 的原创思想获正式命名。
2026 年 2 月
Karpathy 提出"Agentic Engineering"概念 概念
与 Harness 工程形成互补：前者关注如何设计约束与反馈系统，后者关注如何为多 Agent 协同构建工程方法论。行业开始形成新的工程师角色共识。
现在（2026 年 5 月）— Harness 工程实践仍在快速演化，行业规范持续形成中
